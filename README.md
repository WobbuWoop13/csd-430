## Overview
This repository contains selected coursework from **CSD 430: Server Side Development**, organized to demonstrate progressive mastery of **Java-based server-side web development**. The projects focus on building dynamic, database-backed web applications using **JavaServer Pages (JSP)**, **JDBC**, and **MySQL**, following MVC-style architectural patterns.

Rather than serving as a single deployable application, this repository highlights **incremental development milestones**, each emphasizing a core backend concept used in traditional enterprise Java environments.
